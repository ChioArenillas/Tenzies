# Tenzies

## 🧩 Description

Tenzies is a web-based dice game where the goal is to roll until all dice show the same number. Users can hold dice to keep their current values and continue rolling the rest.

The project focuses on game logic, randomization, state tracking, and providing an interactive and engaging user experience.

## 🚀 Live Demo

https://tenzies-delta-one.vercel.app/

## 🚀 Technologies

* HTML5
* CSS3
* JavaScript (ES6+)
* Vercel (deployment)

## 🎯 Features

* Roll dice and hold selected ones
* Track number of rolls
* Dynamic updates of dice values
* Interactive and responsive UI
* Lightweight and fast performance

## 🧠 Key Concepts

* Game logic implementation
* Random number generation
* DOM manipulation
* Event handling
* State tracking without frameworks
